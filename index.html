Game Tips & Tricks

  • Use the tilde key to open the console command pane.
  • If there are visual errors in gameplay, change the game's graphics to anti-alias.

Notes:

  • DBGConsoleOn=true
  • You will need to modify the games general.ini file with the above code to input console commands.
